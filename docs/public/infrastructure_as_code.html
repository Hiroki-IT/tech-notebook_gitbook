・Ansible

Ansibleでは，ymlの文法を用いて関数処理を実行できる．

ファイル名 役割
playbook.yml ソフトウェアのインストールタスクの手順
inventory/* 反映先のサーバの情報
group_vars/* 複数のサーバへの設定
host_vars/* 単一のサーバへの設定
